# Getting started

Install [node.js](https://nodejs.org) then install project dependencies:

```sh
$ npm install
```

or if using [yarn](https://yarnpkg.com) (suggested)

```sh
$ yarn
```

# Replacing svg

- Add `.svg` class to the `<svg>` element: `<svg class="svg"...>`

# Build JS

Edit *.src.js files, then just:

```sh
$ yarn build
```
